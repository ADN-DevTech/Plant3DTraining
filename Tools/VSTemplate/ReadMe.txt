------------------------------------------------------------------------------------------------
About
------------------------------------------------------------------------------------------------
This template works for Plant 3D 2015, installed at the default paths.

Designed for C# projects using .NET 4.5

It creates a simple project with one command, add the main required references for AutoCAD,
P&ID and Plant3D. Also prepares a PackageContents.xml using the Autoloader format. All files
are copied to AppData\ApplicationPlugins folder on BUILD. The same folder is deleted on CLEAN. 

------------------------------------------------------------------------------------------------
How to Install
------------------------------------------------------------------------------------------------
1. Copy this ZIP file to [My Documents]\Visual Studio 2012\Templates\ProjectTemplates\Visual C#
   ***DO NOT unzip***
2. On Visual Studio, the new project template should appear
